# Power BI Custom Paginated Table

Based on the existing work of [https://github.com/jeroenterheerdt/PaginatedTablePBIVisual](https://github.com/jeroenterheerdt/PaginatedTablePBIVisual)

For more information on developing a custom visual see [official tutorial](https://github.com/Microsoft/PowerBI-visuals-docs#developing-your-first-powerbi-visual).

Once you have cloned this example, run these commands to install dependencies and to connect the visual into powerbi.
```
npm i

pbiviz start
```
